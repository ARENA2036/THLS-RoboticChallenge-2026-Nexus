# THLS-RoboticChallenge-2026-Nexus
Results of the team "Nexus" for the Robotic Challenge 2026.

Automated wire harness production is constrained by fragmented design inputs (PDF, KBL, VEC, STP), which describe geometry and BOM but lack a machine-readable manu-
facturing blueprint. As a result, process planning remains largely manual, variant-specific, and dependent on expert knowledge, preventing automation. Stricter design standardization or au-
tomation of isolated process steps addresses symptoms and reduces complexity locally, but fails to enable an integrated, end-to-end automated production system. 

This work proposes a solution to automatically derive BOM, geometry, and a Bill of Process directly from engineering design data, transforming engineering data into structured, robot-ready process descriptions across all
production stages. The benefits of this software-driven approach enables end-to-end integration of multiple robotic assembly steps, significantly reducing engineering effort and ramp-up time while improving scalability, automation robustness, and production efficiency – laying the foundation for cost and time reduction in future automated wire harness manufacturing.
